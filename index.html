Index of Systo web pages

Examples

Page Description
auto_layout.html Demonstrates the use of an automatic layout library for node-and-arc diagrams.
The example provided uses an existing (well-laid-out_ Systo model, but the approach can be used for purely text-based models, without any prior positioning of nodes.
basic.html This is a very cut-down Systo demonstration page. It is intended for people wishing to make their own page to see the basics of how to do this.
diagram_svg.html This example illustrates the use of an alternative, SVG-based, diagramming widget. Currently, this is only for static display of a model diagram.
example_ian.html This example shows how attributes of complex shapes, expressed in SVG, can be linked to modl variables. It uses symbols from the IAN (Integration and Application Network) symbol library.
Note: This example must be run using a local server!

Page Description
pagemaker.html This page allows you to layout and re-size widgets by pacing them using the mouse on a blank window. Potentially this could then be saved as HTML, to make a custom Systo web page, but this has not been done yet.
Compare with sysiwyg.html.
sysiwyg.html This page provides a pretty full WYSIWYG (what you see is what you get) editor, whioch also allows you to plave Systo widgets within the page.
Compare with pagemaker.html.
